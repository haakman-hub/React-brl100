# Koeltechniekmonteurs

This project is a Progressive Web App (PWA) for koeltechniekmonteurs, equipped with offline-first capabilities, IndexedDB support via Dexie, Google Drive sync functionalities, and essential UI components.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/haakman-hub/React-brl100.git
   cd React-brl100
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm run dev
   ```

## Features
- Offline-first behavior
- Sync with Google Drive
- PDF generation

## Development
This project utilizes Vite as the build tool. To learn more about configuring Vite, check their documentation.
